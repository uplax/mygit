Транзакции - На вкладке транзакций выводится список всех операций по всем счетам, а так же форма для проводки новой транзакции. Таблица интерактивная, с быстрым поиском по всем ячейкам, а так же реализованны ссылки на счета и контрагента, указанных в операции.

ВНИМАНИЕ!!! Невозможно провести транзакцию с пустыми значениями дебета, кредита, контрагента.

Контрагенты - на данный момент, работают без интерактивного режима. Ищу способ тянуть всю нужную информацию прямо с налоговой (по ИНН).

Рабочая группа счетов - позволяет использовать в работе только нужные счета.

ВНИМАНИЕ!!! При удалении счета из рабочей группы,настоятельно рекомендуется убедиться, что по нему нет транзакций. Транзакции будут удалены вместе со счетом.

Отчеты - Формирует именную выписку по выбранным счетам в таблицу EXCEL. Отчет можно сохранить на диск или удалить.
